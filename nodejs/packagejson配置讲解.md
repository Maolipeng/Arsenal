
# dependencies区别
## dependencies
依赖关系，组件所需要的, 使用`npm install --save` 或 `npm i -S`

## devDependencies
使用`npm install --save-dev` 或者 `npm i -D`, 开发过程所需要的，外部使用者不用关心的组件。比如单元测试，预编译处理
